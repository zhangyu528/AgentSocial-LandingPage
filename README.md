# AgentSocial Landing Page

This is the landing page for [AgentSocial](https://agentsocial.dev), a bridge between terminal-bound AI agents and social collaboration platforms.

## Installation

To install the AgentSocial CLI, run:

```bash
npm install -g @zhangyu528/agentsocial -- @zhangyu528:registry=https://npm.pkg.github.com
```

## Development

This is a [Next.js](https://nextjs.org) project.

First, run the development server:

```bash
npm run dev
```

Open [http://localhost:3000](http://localhost:3000) with your browser to see the result.

## Tech Stack

- Next.js 16 (App Router)
- React 19
- Tailwind CSS 4
- TypeScript
